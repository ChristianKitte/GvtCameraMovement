EA 5 - Kamerabewegung

Im Rahmen der fünften Einsendeaufgabe sollen zunächst zwei Grundkörper, sowie eine rekursive Kugel erstellt werden. Die Tiefe der Rekursion soll frei einstellbar, die Darstellung der Linien optional sein.

Die Körper sollen frei betrachtet werden können. Hierzu kann mit der Pfeiltasten die Kamera kreisen. Mit den WASD Tasten kann die Kamera in XY Richtung bewegt werden. Mit der N (Shift-N) Taste kann die Entfernung der Kamera eingestellt werden. Der gesamte Code kann auf GitHub eingesehen werden .

Mit Hilfe des Button "Reset" kann der Anfangszustand wiederhergestellt werden. Über die Checkbox kann die Anzeige der Linien gesteuert werden.

Aktuell wird die Figur x angezeigt
